Bookmark Organizer

Live App Client repo Server repo

An app to make bookmarks more useful. Users can upload a bookmarks export file or add bookmarks individually. The app allows sorting, searching, and tagging the bookmarks, and also provides a live preview option and archive search for dead bookmarks. With an account multiple lists of bookmarks can be stored. This was a team project. I handled most of the server-side work and some functionality for the client.

Fresh Tracks

Live App Client repo Server repo

A simple app for tracking winter sports activity. Solo project. With an account, users can provide information about their winter sports activity (dates, location, vert gained, etc), and view summaries by time period or the individual records.
