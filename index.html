Quiz Hunter

Live App GitHub repository

A simple app for a quiz about the Monster Hunter series. The back end is built as a generic quiz app, though since scoring is handled locally it is only suitable for entertainment purposes. The app is designed to scale to any number of questions. Questions and answers are presented in randomized order so that users wishing to improve their score must learn the answer instead of a pattern.
